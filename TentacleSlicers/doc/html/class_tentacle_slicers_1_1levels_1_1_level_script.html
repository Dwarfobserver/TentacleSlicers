Detailed Description

Définit le déroulement d'un niveau via une file d'étapes, qui constituent des vagues. Chaque étape constitue une nouvelle vague si son délai n'est pas nul, sinon elle s'ajoute à la vague courante. Le score total du niveau est calculé.
