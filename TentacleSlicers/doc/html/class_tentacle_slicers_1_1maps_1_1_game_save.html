Detailed Description

Stocke les sauvegardes existantes et permet de créer et charger des sauvegardes. Les sauvegardes conservent l'état de la partie du début du niveau. La création d'une sauvegarde est détaillé dans le dossier resources/saves avec le fichier reamde.
