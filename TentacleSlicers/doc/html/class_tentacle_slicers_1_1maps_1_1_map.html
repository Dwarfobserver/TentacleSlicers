Detailed Description

Représente les données chargées par les fichiers .map, qui n'évolueront pas au cours d'une partie : Les textures de la map, ses murs et collisions, ses deux points d'apparition pour les joueurs et ceux de leurs adversaires. La création d'une map est détaillée dans le dossier resources/maps avec le fichier readme.
