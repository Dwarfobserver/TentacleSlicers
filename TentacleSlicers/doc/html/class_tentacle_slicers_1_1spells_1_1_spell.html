Detailed Description

Un sort (ou compétence) peut être utilisée par un acteur, qui va alors l'incanter pour déclencher son effet. Un sort peut posséder beaucoup de caractéristiques, données par la classe SpellData qui lui est fournie lors de son instanciation.
