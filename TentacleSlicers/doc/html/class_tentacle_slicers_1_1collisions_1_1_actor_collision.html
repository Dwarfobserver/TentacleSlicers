◆ UpdateAndCollide()

void TentacleSlicers.collisions.ActorCollision.UpdateAndCollide ( )

Appelé lorsque l'acteur se déplace afin de l'actualiser dans le gestionnaire de collision et appliquer tous les possibles effets de collision. Pour chaque acteur entré en collision, si celui-ci n'est pas mort, les effets de collision sont appliqués à cet acteur et les effets de collision de cet acteur sont appliqué à l'acteur associé. Enfin, selon le type de hitbox de chaque acteur, un des deux acteurs peut être replacé. En cas de deux types de hitbox de même type (non immaterielles ou unmovables), c'est l'acteur en cours de déplacement qui est replacé.
