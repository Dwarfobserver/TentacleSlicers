Detailed Description

Définit une vague du niveau, avec l'apparition de plusieurs mobs identiques à un point. Permet de s'ajouter à la vague précédente si le délai de ce LevelStep est nul.

Crée un spawn de plusieurs mobs du type donné pour l'endroit donné, après un délai dans le déroulement du niveau. Si ce délai est nul, alors cette étape fera parti de la même vague que l'étape précédente.

Parameters
secondsTimerLe délai pour lancer l'étape, en secondes
mobTypeLes données servant à créer les mobs invoqués
nbMobLe nombre de mobs à invoquer
spawnL'endroit où sont invoqués les mobs
