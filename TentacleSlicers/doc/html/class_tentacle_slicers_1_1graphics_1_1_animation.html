Detailed Description

Charge les images d'un dossier et les associe à un type d'animation. Les images du dossier doivent être nommées par leur ordre (de 0 au nombre d'images - 1) et au format png. De plus, le dossier doit contenir les sous-dossiers top, bottom, left et right (avec le même nombre d'images) si l'animation dépend d'une orientation.
